Supporting

The NIST/CDC CDA Test Suite supports a broad range of testing in support of the HL7 Vital Records Community, including transport, messaging (content), and functional.

  • NCPDP Structured SIG Validation
  • NCPDP XML Validation
  • NCPDP SCRIPT Implementation Guide Release 2016071
Structured SIG Testing Start Testing

The NIST/NCPDP ePrescribing Test Suite is the test tool used for testing the SCRIPT standrard. The capability used for NCPDP Certification can be accessed by navigating to the NCPDP Certification Test Plan found on the Context-based tab.

Have a Question?

A Google Group (ePrescribing Testing) has been established for discussion/questions about the Implementation Guide, testing tool, and testing issues. No membership is required. A Google account is required for posting. Note: The same Google Group is being used for submitting questions/comments related to the 2014 and 2015 Editions of the NIST Certification Test Tool and general ePrescribing related topics. When asking questions, please specify the edition of the tool and version of the standard to which the questions apply.

  • Site: https://groups.google.com/d/forum/erx-testing-tool
  • Email: erx-testing-tool@googlegroups.com
